## E-Wallet Onboarding Project

Technical using

```bash
1. ReactJS
2. Typescript
3. Eslint, Prettierrc
4. Craco
5, Scss
```

Get node_modules

```bash
yarn
```

Run the development server:

```bash
yarn start
```

Open [http://localhost:3030](http://localhost:3000) with your browser to see the result.
